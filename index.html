Prerequisites

  • Storage: At least 15GB (testnet) / 2TB (mainnet) of free disk space (preferably SSD)
  • Memory: 4GB RAM minimum (8GB recommended)
  • Internet: Stable broadband connection for initial blockchain download
  • Time: 2-4 hours for complete setup (depending on internet speed)
  • Basic Skills: Comfort with downloading software and using terminal/command line

Overview of the process

  1. Initial Setup - Create workspace and organize files
  2. Download Binaries - Bitcoin Core, Fulcrum & Sparrow Wallet - Get official software and verification files
  3. Verify Downloads - Check checksums and developer signatures
  4. Install & Configure - Set up Bitcoin Core, Fulcrum and Sparrow Wallet with proper configuration
  5. Blockchain Sync - Download and verify the Bitcoin blockchain
  6. Testing & Validation - Ensure your connection between node -> electrum server -> wallet
  7. Wallet & Transaction - Create a wallet and make testnet transactions
